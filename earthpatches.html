Earth Patch Notes

Outside is the most played video game of all time. With thousands of years of developer support, over 8 billion active players and almost 200 active servers, it is a massive hit among all demograpnics. Throughout centuries several new story modes were added, new quests continue to be developed and the minigame development community is the most active.

Below we present Earth patch notes, starting from v2.0.24-7

Earth Patch Notes v2.0.24-8c4
  • Fixed some sirens not working during the Weewoo Event (Netherlands server)
  • Removed Herobrine
  • Fixed some minor bugs with Nintendo 3DS family items
  • Adjusted fly mob spawn rate
  • Cat companions should no longer spawn in high places at player houses
Known bugs and issues
  • A bug is currently affecting the Grand Exchange on USA server where players lose their money due to an unknown error in the economy code. We are currently investigating the cause.
  • Due to a player riot crisis on UK server, players are advised to not connect or migrate account data there until further notice
Earth Patch Notes v2.0.24-8c3
  • Adjusted mosquito spawn rate in some servers
  • Removed Herobrine
  • Europe Server Cluster staff is now gathering player feedback on minigames lifespan and maintenance - see more information here
  • Pride mini-event has started in Netherlands server
  • Every player with a cat companion receives a temporary +50 happiness buff every Saturday
  • Minor adjustments made on server connectivity
  • “Instagram” service has been temporarily discontinued on Turkey server
Known bugs and issues
  • It is still hard to migrate account data between servers, even within the Europe Server Cluster. We are working hard on adjusting the protocol.
  • Unclaimed cat companions still have a chance of randomly despawning even when befriended by a player
  • The latest update (v2.0.24-8c2) involving new player gender system in Germany server puts players in a 3-month cooldown before changes take effect.
  • Transport system in Germany server still has several delay/cancellation bugs that we try ironing out
The developers of Outside are reminding all USA server players to register for new head staff voting and to check if their registration is valid.
